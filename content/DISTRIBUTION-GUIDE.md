# Distribution Guide — minimum-effort traffic for mayaallan.com

**Your time constraint:** ~30 minutes to set up, then ~15 minutes per post after drafts are written.

**Your goal:** people who need the tools actually find them.

**What AI has already done:** drafted 3 substantive posts, wired analytics into the site so you can see what works, built the three tools, set up the site's SEO metadata for good social-share previews.

**What only you can do:** decide whether to publish them, answer the occasional genuine reply, and not quit after 6 posts when it looks like nothing is happening.

## Step 1 — One-time setup (30 minutes total, do once)

### A. Substack account (10 min)

1. Go to [substack.com](https://substack.com) → Sign up
2. Use the email you want readers to reach you at
3. Publication name suggestion: **"Maya Allan"** or **"Belief Systems Explorer"** (your call — first is simpler)
4. Publication URL: e.g., `mayaallan.substack.com`
5. Upload a simple square photo (reuse your book author headshot)
6. Description: *"Research-backed writing on belief systems, nervous system regulation, and psychedelic integration. By author Maya Allan — explorer, not authority."*
7. Turn OFF: Substack Notes, comments (or set to paid-only, which at $0 means none), live video prompts
8. Turn ON: recommendations (so other writers can promote you)
9. **Done.** Close the tab.

### B. Amazon Author Central (10 min)

1. [authorcentral.amazon.com](https://authorcentral.amazon.com) → sign in with your Amazon account
2. Add your books if not already (search by ISBN — it auto-pulls)
3. Update your **Author Bio**: include the sentence "Find free reflection tools at mayaallan.com."
4. Under "Author Page": link your website
5. **Done.** Every future book buyer on Amazon now sees your website in your author profile.

### B2. Other author-profile platforms (55 min total, all one-time, all free)

These are "set up and forget" — they work in the background forever once set up. Each is a backlink to mayaallan.com and a way readers find you.

**Goodreads Author Program** (20 min) — Largest book community online (~90M readers).
1. [goodreads.com/author/program](https://www.goodreads.com/author/program) → Apply
2. Claim your book profile
3. Add bio, photo, link to mayaallan.com, link to Substack (when you have it)
4. Approval in a few days

**BookBub Author Profile** (15 min + 1-2 week approval) — 15M+ reader newsletter.
1. [partners.bookbub.com](https://partners.bookbub.com) → Apply as Author
2. Submit your book's Amazon ASIN for verification
3. Add bio, photo, website
4. **Important:** free profile only. Don't pay for Featured Deals yet — they're $150-$2000 and only worth it at scale.

**Open Library** (10 min) — Internet Archive's catalog. High-authority SEO backlink.
1. [openlibrary.org/account/create](https://openlibrary.org/account/create)
2. Search for your book → click author name → Edit author page
3. Add bio + photo + website + links to your other books

**LibraryThing** (10 min) — Smaller but dedicated reader community.
1. [librarything.com/signup.php](https://www.librarything.com/signup.php)
2. Claim your author page
3. Same drill: bio, photo, website link

**Why bother with all five?** Each is a one-time 10-20 min effort that drives traffic forever. Readers browsing *any* of these platforms find you. Collectively they build domain authority for mayaallan.com in Google's eyes. Zero ongoing work after setup.

### C. Pinterest account (10 min — optional but worth it for your niche)

Pinterest has huge long-tail traffic for "journaling prompts," "inner work," "shadow work," "nervous system regulation" — your exact territory.

1. [pinterest.com/business/create](https://pinterest.com/business/create) — choose Business account (free, has analytics)
2. Display name: Maya Allan
3. Add profile photo + bio mentioning mayaallan.com
4. Create one board: *"Belief Work & Inner Exploration"*
5. **Done.** Pins from your site will now show up in Pinterest search.

## Step 2 — Per post (15 min, every 1-2 weeks)

You have 3 drafted posts in `content/posts/`. Each one publishes in ~15 minutes of your time.

### For each post:

1. **Open the draft markdown file** in `content/posts/`
2. **Skim the post** — make sure every sentence sounds like you would say it. Edit any line that doesn't. (If three lines feel off, rewrite them. Don't rewrite the whole thing — the research citations, the structure, the argument all hold together.)
3. **On Substack:**
   - New post → paste the markdown
   - Substack's editor accepts markdown natively (headings, bold, blockquotes, footnotes all render)
   - Upload a simple cover image — you can use Unsplash (free), search e.g. "meditation" or "introspection"
   - Title and subtitle are in the frontmatter at the top of the file — copy them into the Substack title/subtitle fields, then delete the frontmatter section from the body
   - Tags: Substack allows up to 5 — use the ones in the frontmatter
   - **Schedule** for Tuesday or Thursday morning (9-11 a.m. EST, the best open rates for newsletters)
   - Click Send
4. **On Pinterest** (if you set it up):
   - Create a simple 1080x1920 vertical image in Canva (free) with the post title and your site URL
   - Create Pin → upload → destination URL = the Substack post → title = post title → 2-3 sentences of description
   - Save to the "Belief Work" board
5. **On LinkedIn** (optional — 5 min):
   - Post the first paragraph + "Read the full piece →" with link
   - LinkedIn's algorithm rewards long-form writing in wellness/psychology niches
6. **In your site's home page** — link to the post from the Resources section (the content/posts files can eventually become blog posts on the site itself — ask AI to do this)

That's it. Close the tabs. Go back to real estate.

## Step 3 — Let it compound (3-6 months, no action)

Here's what you should expect:

- **Weeks 1-4:** Post 1 gets 10-30 views. You feel like nothing is working. This is normal. Keep going.
- **Weeks 4-8:** Google starts indexing. Search traffic trickles. Substack Recommendations (if you've reached out to 5-10 aligned writers) start to matter. 50-100 views per post.
- **Months 3-6:** Compound: backlog of 10-12 posts is all indexable, each pulling long-tail search queries. One post might catch fire on Reddit (shared by someone else, not you) or get linked by a larger writer. Expect one post in 20 to 3-10x the others.
- **Months 6-12:** Your Substack is an asset. 200-500+ subscribers is realistic for a researched belief-work niche with genuine writing. Podcasters start reaching out to you for guest spots — which was impossible at month 0.

**The single failure mode:** quitting at month 2 because nothing has happened yet. It always happens. Just keep shipping the drafts AI writes.

## Step 4 — Growth multipliers (when you want to, in 12-24 months)

Only worth doing AFTER you have 5-10 published posts and ~100 subscribers. Not now.

1. **Substack Recommendations** — pick 5-10 writers whose Substacks feel adjacent (mindfulness, wellness, mental health, integration coaches). Subscribe to theirs. After 6 weeks, email them genuinely with the ones you found most useful — mention you'd love them to consider recommending yours, one-time ask. Expect 1 in 5 to say yes. Those recommendations drive real subscriber growth.

2. **Guest posts** — once you have a portfolio on Substack, other publications (not podcasts — writing) will accept pitches. This is how you get in front of other people's audiences without being on their podcast.

3. **Instagram Reels** (only if you enjoy it) — one 30-second reel per post, explaining the core idea. 1-3 times a week. Algorithm-friendly, doesn't require you to be "on" personally. Script-to-camera works; you don't need to be charismatic.

4. **Facebook Groups** — there are active wellness/integration groups where genuine content performs well. Same rule as Reddit: be a member, contribute, occasionally share your own work when it genuinely fits the thread. 30 min/month.

## Medium cross-posting (wired, ready to use)

Every blog post on your site can now be auto-copied to Medium as a **draft** (with a canonical link pointing back to mayaallan.com, so Google treats your site as the original). You review in Medium's editor, hit Publish when ready.

### One-time setup (5 min)

1. **Create a Medium account** at [medium.com](https://medium.com) if you don't have one
2. **Generate an integration token**: [medium.com/me/settings/security](https://medium.com/me/settings/security) → Integration tokens → enter description like "mayaallan-cross-post" → Get token
3. **Copy the token** — it's only shown once. Looks like `2abc...` (long string).
4. **Add to your shell env** (or prefix on each command):
   - Windows PowerShell: `$env:MEDIUM_INTEGRATION_TOKEN = "your-token-here"`
   - Mac/Linux: `export MEDIUM_INTEGRATION_TOKEN="your-token-here"`

### To cross-post a post (one command per post)

From the project directory:

```bash
pnpm publish:medium audit-is-the-wrong-word
```

The slug is everything after the number prefix in the filename. `01-audit-is-the-wrong-word.md` → slug is `audit-is-the-wrong-word`.

The script:
1. Reads the markdown file
2. Posts to Medium via their API as a **draft**
3. Sets canonical URL to `https://www.mayaallan.com/blog/audit-is-the-wrong-word` (so your site keeps the SEO)
4. Prints the Medium draft URL for you to open and review

### To publish immediately instead of as draft

```bash
pnpm publish:medium audit-is-the-wrong-word --public
```

I'd recommend always using the default (draft mode) and doing a quick review in Medium's editor before publishing. Takes ~2 min.

### Important notes

- Medium's API has been known to rate-limit. Don't cross-post more than 1-2 posts in quick succession.
- If Medium changes their API terms or revokes your integration token (rare but possible), you can always fall back to manual copy-paste. The `content/posts/` files are always your source of truth.
- For Substack, you still copy-paste manually — Substack has no official publishing API.
